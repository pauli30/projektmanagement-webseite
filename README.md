# Mode Wiederverkaufsplattform

Eine einfache HTML/CSS/JavaScript Website für eine Mode Wiederverkaufsplattform.

## Funktionen

- Startseite mit ausgewählten Produkten
- Produktseite
- Warenkorb (grundlegend)
- Anmeldeseite
- Kundenkonto zur Anzeige des Projektmanagementplans

## Erste Schritte

Öffnen Sie `index.html` in Ihrem Webbrowser, um die Website anzuzeigen. Für GitHub Pages ist die Seite jetzt in `docs/` verschoben.

## Struktur

- `docs/index.html`: Startseite
- `docs/css/styles.css`: Stylesheet
- `docs/js/script.js`: JavaScript für Interaktivität
- `docs/pages/`: Zusätzliche Seiten
- `docs/images/`: Platzhalter für Produktbilder
- `docs/files/projektplan.pdf`: PDF mit Projektplan

Hinweis: Ersetzen Sie `images/placeholder.jpg` durch tatsächliche Produktbilder.
